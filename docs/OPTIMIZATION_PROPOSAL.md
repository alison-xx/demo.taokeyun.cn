# 灵策智算 - 系统优化与迭代方案 (v2.0)

## 1. 架构升级 (前端)
- **当前状态**: 原生 HTML + JS + CSS，代码全在一个 `main.js` (184KB) 中，难以维护。
- **优化方案**: 全面迁移至 **Vue 3 + TypeScript + Vite**。
- **收益**: 组件化开发、强类型校验、更好的状态管理 (Pinia)、极速的热更新体验，大幅提升后续迭代效率。

## 2. 架构升级 (后端)
- **当前状态**: 原生 PHP，流式输出 (SSE) 占用阻塞进程，高并发能力弱。
- **优化方案**: 迁移至 **Node.js + Express + TypeScript**。
- **收益**: 原生异步非阻塞 I/O，完美契合 AI 的流式对话 (SSE) 场景，前后端语言统一 (TypeScript)，降低维护成本。

## 3. 核心功能重构与补充
- **多模态与 AI 绘画**: 接入完整的图像生成 API（如 SiliconFlow），支持文生图、图生图。
- **多 Agent 协作**: 真正实现后端的 Agent 编排，支持“组合回答”和“分开回答”。
- **Markdown 与代码高亮**: 前端引入 Markdown 渲染引擎和 Highlight.js，提升程序员和专业用户体验。
- **状态同步与持久化**: 使用 Pinia 和 LocalStorage/Cookie 进行可靠的会话状态管理，支持多 Tab 状态同步。

## 4. 安全与性能防护
- **CORS 与跨域**: 严格配置允许的跨域域名。
- **接口限流 (Rate Limiting)**: 引入限流中间件，防止恶意刷 Token 与配额耗尽。
- **统一错误处理**: 全局拦截 API 错误并提供友好的 UI 提示 (Toast/Message)。

## 5. 实施计划 (Roadmap)
- **阶段一**: 备份旧版原生代码，初始化全新 Vue3 + Node.js 工程。
- **阶段二**: 迁移数据库模型与核心用户认证 (Auth) 流程。
- **阶段三**: 重构前端 UI 框架与状态管理，迁移 AI 员工列表。
- **阶段四**: 重构核心 AI 聊天界面与后端 SSE 流式接口，实现基础对话。
- **阶段五**: 补齐 AI 绘画、个人设置、历史记录与后台管理等周边功能。
